# Micro-Friend
<h3>게기스 제4회 게임잼 대상작</h3>
<h5>설명</h5>

우주선을 목적지까지 이동시켜야하는 2인용 협동게임

우주선 내부의 설비는 특정 플레이어만 작동이 가능하기 때문에 협동이 필요하다.

<h5>조작</h5>

1P : 주황색

이동 : WASD , 점프 : F , 작동 : G

2P : 초록색

이동 : 방향키 , 점프 : K , 작동 : L
